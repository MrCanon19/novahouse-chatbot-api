# Symulowany Projekt Google Cloud Platform

Ten katalog symuluje utworzenie nowego projektu Google Cloud Platform dla NovaHouse. 

**Kluczowe informacje z analizy projektu:**
- NovaHouse posiada jedynie Google Drive, co oznacza konieczność założenia nowego konta Google Cloud Platform.
- Michał Marini będzie administratorem projektu po zakończeniu wdrożenia.
- Maksymalny miesięczny budżet na usługi Google Cloud: do 400 zł/mc.

**Następne kroki (symulowane):**
1. Aktywacja niezbędnych API (np. Dialogflow API, Cloud Functions API).
2. Konfiguracja uwierzytelniania (Service Account).
3. Ustawienie budżetu i alertów.
